# Better Health Indicators

A lightweight Fabric mod that displays player health above their heads using a clean vanilla-style heart display.

## Features

* ❤️ Displays player health above their heads
* 🎮 Works on multiplayer servers
* 🪶 Lightweight and client-side only
* 🎨 Uses vanilla-style heart textures
* ⚡ Real-time health updates


## Requirements

* Minecraft 26.1.X
* Fabric Loader
* Fabric API

## Compatibility

This mod is client-side only and does not require installation on the server.

## Building

Clone the repository and run:

```bash
./gradlew build
```

The built jar will be located in:

```text
build/libs/
```
